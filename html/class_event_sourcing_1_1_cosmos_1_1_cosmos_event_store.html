Add Eventss to the IEventStore<TBaseEvent>

When adding events an EventStoreException will occur when

  • An Events with the same Event.Id already exists
  • An Events with the same combination of Event.AggregateId and Event.AggregateVersion already exists
  • The added Eventss do not all share the same Event.AggregateId
